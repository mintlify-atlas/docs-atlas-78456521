# Welcome to Zed

Zed is a **high-performance, multiplayer code editor** from the creators of [Atom](https://github.com/atom/atom) and [Tree-sitter](https://github.com/tree-sitter/tree-sitter). Built with Rust and GPU-accelerated rendering, Zed delivers native performance with a modern editing experience.

## Why Zed?

### Lightning Fast

Zed uses Metal (macOS), Vulkan (Linux), and DirectX 11 (Windows) for GPU-accelerated rendering. Written in Rust, it starts instantly and handles large codebases with ease.

### Built-in AI

Zed includes native AI features:

- **Agentic editing** - AI agents that read, write, and refactor code
- **Edit prediction** - AI code completions on every keystroke
- **Inline assistant** - Transform code selections with AI
- **Multi-model support** - Use Anthropic, OpenAI, Google, Ollama, and more

See the [AI Overview](https://zed.dev/docs/ai/overview) to learn more.

### Real-Time Collaboration

Multiple developers can work in the same project simultaneously, seeing each other's cursors and edits as they happen. Voice chat is built-in.

- **Channels** - Persistent project rooms for teams
- **Private calls** - Ad-hoc collaboration with contacts
- **Shared projects** - Everyone edits the same codebase in real-time

Learn more about [Collaboration](https://zed.dev/docs/collaboration/overview).

### Powerful Terminal

Zed's integrated terminal supports:

- Multiple terminal instances and tabs
- Custom shells and environment configuration
- Deep editor integration
- Split panes

Read the [Terminal documentation](https://zed.dev/docs/terminal).

### Extensive Language Support

Zed supports hundreds of programming languages. Many are built-in, while others are available through extensions:

**Built-in languages include:**
- Rust (Zed is written in Rust)
- JavaScript, TypeScript, JSX, TSX
- Python, Go, C, C++
- JSON, YAML, Markdown, HTML, CSS
- And many more

**Extension-based languages include:**
- Java, Kotlin, Scala
- Ruby, Elixir, PHP
- Haskell, OCaml, Elm
- Swift, Dart, Zig
- And 100+ community extensions

Browse all [supported languages](https://zed.dev/docs/languages).
